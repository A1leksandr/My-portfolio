<ol>
- Задача 1: собрать калькулятор юнит-экономики с целями на следующий год:
<p>- выйти на 4000 активных студентов,</p>
<p>- удержать маржинальность на уровне выше 15%,</p>
<p>- выйти на выручку выше 35 000 000 руб. в месяц,</p>
